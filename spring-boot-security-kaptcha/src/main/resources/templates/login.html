对于外界普遍关注的新区如何建设、新区如何吸引人才等问题，根据纪实披露，习近平总书记曾明言，新区的规划发展要以人为本，人往高处走，水往低处流，公共服务水平高的地方更具吸引力。国家发改委主任何立峰则表示，新区的基本公共服务水平甚至可能会高于现在的北京。

今年2月23日，中共中央总书记、国家主席、中央军委主席习近平专程来到河北省保定市安新县，实地察看规划新区核心区概貌。在大王镇的田野上，在展开的一张规划图前，总书记仔细察看区位、规划状况，详细了解人口搬迁安置等情况。据了解，新区的人口搬迁主要是就地安置。
